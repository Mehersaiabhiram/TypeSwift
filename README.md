# ⌨️ TypeSwift - Typing Speed Test

TypeSwift is a modern, responsive typing speed test web application built using **HTML**, **CSS**, and **JavaScript**. It helps users improve their typing speed and accuracy by displaying random motivational quotes and providing real-time performance statistics such as Words Per Minute (WPM), accuracy, and elapsed time.

---

## 🚀 Features

- ⌨️ Real-time Typing Test
- ⚡ Live Words Per Minute (WPM) Calculation
- 🎯 Accuracy Percentage Tracking
- ⏱️ Real-Time Timer
- 📝 Random Motivational Quotes
- ✅ Character-by-Character Validation
- 🔄 Reset & Retry Functionality
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- 🚀 Lightweight and Fast
- 🌐 Works on Desktop and Mobile Devices

## ⚙️ How It Works

1. Click the **Start** button.
2. A random motivational quote is displayed.
3. Start typing the quote.
4. The application tracks:
   - Time
   - Words Per Minute (WPM)
   - Accuracy
5. Once completed, your final performance is displayed.
6. Click **Reset** to start a new test.
